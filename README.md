# Prime Numbers

Simple program to calculate the sequence of prime numbers.

## Dependencies

- [GMPlib](https://gmplib.org/)

Arch:

```bash
pacman -S gmp
```

## Compile

with make and gcc:

```sh
make
```

## Params

- -r (number)     --rank (number)         Get the max rank of the suite.
- -js             --json                  Use json format.
